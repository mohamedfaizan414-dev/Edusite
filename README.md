Edusite
Edusite is a comprehensive educational learning website designed to provide structured content across multiple academic disciplines. Built primarily with HTML, this platform serves as a centralized hub for students to access resources in Science, Mathematics, English, and History.

🚀 Features
Subject-Specific Hubs: Dedicated sections for Biology, Chemistry, Physics, Mathematics, History, and English.

Detailed Math Modules: Specialized pages for Algebra, Linear Inequalities, and Quadratic Equations.

Historical Deep-Dives: Lessons covering Ancient Egypt, Medieval History, and Modern History.

Progress Tracking: Includes a dedicated progress page to help learners monitor their journey.

Clean Navigation: Multi-page architecture for an organized learning experience.

📂 Project Structure
The repository contains the following key modules:

index.html: The main landing page and navigation hub.

Mathematics: math.html, mathsalgebra.html, maths.linearinequality.html, maths.quadraticequation.html.

Science: science.html, bio.html, chem.html, phy.html.

History: history.html, hisegypt.html, hismedival.html, hismodern.html.

English: english.html, eng1.html, eng2.html, eng3.html.

User Tools: progress.html.
